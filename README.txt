# おうち食材で献立 — Pantry Planner

iPad/スマホ/PC のブラウザでそのまま動く、オフライン対応の献立アプリ（PWA）。  
家にある食材（パントリー）を入れると、一致率でレシピ候補を出して献立表にもドラッグ&ドロップで並べられます。

## 使い方（超ざっくり）
1. `index.html` をブラウザで開く（Safari/Chrome どちらでもOK）。
2. 上部の「インストール」→ホーム画面に追加するとアプリ風に使えます（オフラインOK）。
3. 「家にある食材」に食材名を入れて「追加」。数量メモも書けます。
4. 一致率スライダー・時間・カテゴリ・検索で絞り込み。
5. レシピカード →「詳しく」で作り方、不足食材、コツを表示。
6. 「献立に」で1週間のグリッドに自動配置（最大3つ/日）。削除は ✕。
7. パントリー/献立は自動保存。JSONで書き出し・読み込みも可能。

## データ拡張
- `recipes.json` にレシピを追記すれば候補が増えます。  
  フィールド: `name`, `time`, `category[]`, `ingredients[]`, `optional[]?`, `steps[]?`, `tips?`
- 簡単な同義語（卵↔たまご、ねぎ類、カレー粉/ルウなど）に対応した一致判定を内蔵。
